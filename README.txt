FATIMAZAHRA BIRTHDAY WEBSITE

HOW TO OPEN
1. Extract the ZIP file.
2. Open the extracted folder.
3. Double-click index.html.

MUSIC
Add your chosen song to the same folder as index.html and name it:
music.mp3

PUBLISHING
Upload the entire folder to Netlify Drop, Vercel, or GitHub Pages.
Do not upload only index.html because the images are inside the assets folder.
